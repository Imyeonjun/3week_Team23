# 3weak_Team23 - LoopDungeon

내일배움캠프 Unity 9기 3주차 팀 과제 (23조)

## 🎮 프로젝트 소개
**23 LoopDungeon**은 플레이어가 직업을 선택하고 던전을 탐험하며 몬스터와의 전투를 즐길 수 있는 Text RPG입니다. 
상점과 대장간을 활용해 장비를 강화하며, 여관에서 휴식 후 퀘스트를 받고, 신전에서 버프를 얻을 수 있습니다.

## 🧭 게임의 진행 및 흐름

1. **인트로** : 이름 및 직업 선택
2. **메인 메뉴** : 행동 선택
3. **상점** : 아이템 구매 및 판매
4. **여관** : 퀘스트 수락과 휴식
5. **대장간** : 장비 강화 및 수리
6. **던전** : 전투 진행
7. **신전** : 헌금 및 버프 획득
